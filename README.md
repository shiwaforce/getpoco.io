# getpoco.io

Official website for [getpoco.io](https://github.com/shiwaforce/poco). Built with [Hexo](https://hexo.io).

## Requirements

- **Node.js 20+** (see [.nvmrc](.nvmrc); use `nvm use` if you use nvm)

## Getting started

```bash
npm install
npm start
```

Open http://localhost:4000 in your browser.
